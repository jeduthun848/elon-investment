# FinanceHub - Investment Trading Platform

A professional investment trading platform showcasing portfolio management, stock analysis, and profit tracking.

## Features

- **Live Trading Dashboard** - View real-time portfolio performance
- **Stock Holdings** - Detailed breakdown of investments with gain/loss tracking
- **Educational Content** - Learn how stock trading works and how profits are made
- **Portfolio Analytics** - Comprehensive profit/loss analysis
- **Responsive Design** - Works on desktop and mobile devices

## Pages

- **Home** (`index.html`) - Educational content about stock trading and investment strategies
- **Dashboard** (`dashboard.html`) - Live trading portfolio with detailed analytics

## How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Navigate between Home and Dashboard using the top navigation

## Technologies Used

- HTML5
- CSS3 (with gradients and responsive design)
- No external dependencies

## Deployment

This site is deployed using GitHub Pages and is automatically live from the `main` branch.
